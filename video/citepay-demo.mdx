# CitePay Agent — Demo Presentation

**Public proof viewer:** https://faadil1.github.io/citepay-agent-lepton/index.html

---

## What it is

CitePay Agent is a terminal-based AI agent submitted to the Lepton Agents Hackathon.

It does one thing: before an AI agent can cite a source, it must pay for it.

Payment flows through Circle Gateway x402 on Arc Testnet. The agent scores candidates, rejects the weak ones, pays the winner on-chain, and cites only what it paid for.

The proof viewer above is the **judge presentation layer** — a static page rendering the captured V3 evidence. It is not a production app.

---

## The gate

Three candidate sources. One user query. Fixed budget: 0.005 USDC.

| Source | Price | Relevance | Ratio | Decision |
|---|---|---|---|---|
| S1 — Creator Citation Nanopayments | 0.001 USDC | 0.357 | 357.143 | PAID · CITED |
| S2 — Expensive General Payments Overview | 0.02 USDC | 0 | 0 | REJECTED — over budget |
| S3 — Secondary Creator Monetization Note | 0.001 USDC | 0 | 0 | REJECTED — lower relevance/price |

The agent computed relevance via token overlap between the query and each source's title/content metadata. No LLM call for ranking.
